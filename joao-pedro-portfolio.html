Veja a seguir alguns de meus projetos!

Projeto 1: Script de Automação de Tarefas

Descrição: Desenvolvi um script em Python para automatizar tarefas repetitivas no meu dia a dia, como a organização de arquivos e renomeação de pastas. O objetivo foi simplificar tarefas que consumiam muito tempo, criando um fluxo de trabalho mais eficiente.

Ferramentas Utilizadas: Python, bibliotecas OS e shutil, para manipulação de arquivos e pastas de forma simples e rápida.

Objetivo: Através deste projeto, mostrei minha capacidade de usar a programação para resolver problemas práticos do cotidiano, otimizando processos e tornando a rotina mais eficiente.

Projeto 2: Jogo de Aventura 2D

Descrição: Criei um jogo 2D simples, no estilo pixel art, onde o personagem principal enfrenta desafios e coleta moedas para avançar. O foco foi desenvolver a lógica de programação por trás da jogabilidade e criar uma experiência divertida para os jogadores.

Ferramentas Utilizadas: Godot Engine, GDScript – a escolha da Godot foi para explorar uma engine poderosa, de fácil aprendizado e muito utilizada na comunidade indie de desenvolvedores.

Objetivo: Este projeto teve como objetivo praticar e entender a mecânica de jogos, o comportamento de personagens e o sistema de colisões, criando uma introdução ao mundo do desenvolvimento de jogos e suas várias possibilidades.

Projeto 3: Aplicativo de Rastreamento de Hábitos

Descrição: Este aplicativo foi desenvolvido para ajudar os usuários a monitorar e melhorar seus hábitos diários, seja para saúde, produtividade ou desenvolvimento pessoal. Ele permite registrar e visualizar o progresso ao longo do tempo, incentivando um estilo de vida mais organizado.

Ferramentas Utilizadas: Kotlin e Android Studio – o Kotlin foi escolhido por ser uma linguagem moderna e eficiente para o desenvolvimento de aplicativos Android.

Objetivo: O objetivo deste projeto foi criar uma solução prática para aumentar a produtividade e promover mudanças positivas no comportamento dos usuários, além de aprimorar minhas habilidades em desenvolvimento mobile.

Projeto 4: Web App para Gerenciamento de Projetos

Descrição: Desenvolvi uma aplicação web para gerenciar tarefas e prazos de equipes em projetos colaborativos. O objetivo foi criar uma plataforma acessível e eficiente para controlar o progresso de diversas atividades, ajudando na organização e priorização de tarefas.

Ferramentas Utilizadas: HTML, CSS, JavaScript e Firebase – o Firebase foi utilizado para integrar o banco de dados em tempo real, permitindo que os usuários visualizassem as atualizações de maneira dinâmica e sem necessidade de recarregar a página.

Objetivo: Esse projeto teve como foco a implementação de um sistema completo de gerenciamento de tarefas, com uma interface simples e intuitiva, demonstrando minhas habilidades em desenvolvimento web e integração com bancos de dados na nuvem.
